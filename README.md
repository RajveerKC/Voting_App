# Voting App
This application is a simple GUI application written only using C that simulates a basic electronic voting system.  
The program allows users to vote for candidates and then displays the final voting results.  
This project was developed as a part of a C Programming class project to demonstrate fundamental programming concepts such as:
* Structures
* Arrays
* Functions
* User input/output
* Control Flow

## Build Instructions
**NOTE: Cmake and Git are required to build this project.**
### Generate Makefiles
```
cmake ..
```
### Build the Project
```
cmake --build .
```
